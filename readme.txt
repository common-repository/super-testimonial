== Description ==

Testimonials are easy to use the plugin that allows users to add Testimonials to the sidebar, as a widget, or to embed testimonials into a Page or Post using the shortcode. Testimonials also allow you to insert a list of all Testimonials or output a Random Testimonial. The testimonial is 100% responsive for all devices.

== Installation ==
<strong>Using The WordPress Dashboard</strong>

1. Install as a regular WordPress plugin.
2. Go to your Plugins setting via WordPress Dashboard and activate it.
3. After activating the plugin you will see the "S Testimonials" menu below the comment's menu on the left side of the WordPress dashboard and find "Generate Shortcode".

<br />

<strong>Using FTP</strong>
1. Extract tp-super-testimonials.zip to your computer.
2. Upload the tp-super-testimonials directory to your wp-content/plugins directory.
3. After activating the plugin you will see the "S Testimonials" menu on the left side of the WordPress dashboard and find "Generate Shortcode"

<br />
<strong>How to use short-code?</strong><br />
When active the plugin please Generate a shortcode and used this shortcode in your theme anywhere you want to display this testimonial.
